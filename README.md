## 📚 What I Learned

### Day 01: Digital Foundation
- ได้รู้ว่า SE กับ Solo Dev คืออะไร ต่างกันยังไง และในสายงานนี้เราต้องทำอะไรได้บ้าง
ฟังวิทยากรบรรยายเรื่อง Mobile App และการประกอบอาชีพในสาย Software

### Day 02: Logic & Flowgorithm Lab 1
- เรียนรู้วิธีคิดแบบมีระบบ ต้องคิดยังไงถึงจะเป็น SE ที่ดีได้
ได้รู้พื้นฐานของ Algorithm และการคิดเชิงตรรกะแบบ SE Expert

### Day 03: Flowgorithm Lab 2–3
- เข้าใจว่าโปรแกรมมันทำงานเบื้องหลังยังไง ทำไมมันถึงทำงานได้ตามที่เราต้องการ
ได้ลองเขียน Flowchart เองจริง ๆ และเข้าใจโครงสร้างการตัดสินใจ (If/Else) กับการวนซ้ำ (Loop)

### Day 04: AI, Web Technology and Network
- ได้ฟังเรื่องส่วนประกอบของเว็บไซต์ว่ามีอะไรบ้าง เช่น URL, UX/UI, frontend/backend
ได้ลองทำ Lab เรื่อง Push และ Commit ไฟล์งานขึ้น GitHub ด้วย
